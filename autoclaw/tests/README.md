# Documentación de la Carpeta `tests`

Esta carpeta contiene los casos de prueba para asegurar la calidad y funcionalidad del sistema:
- **Pruebas Unitarias**: Prueban funciones individuales.
- **Pruebas de Integración**: Verifican interacciones entre componentes.

Mantener esta carpeta actualizada con nuevos casos de prueba conforme se desarrollan nuevos módulos.

# Flujo solo con FFmpeg (sin IA)

1. Selecciona una imagen y un audio (puedes usar los de plantillas/imagenes_ejemplo y plantillas/audios_ejemplo).
2. Usa tools/imagen_a_video_ffmpeg.py para crear un video dinámico a partir de la imagen.
3. Si quieres, añade subtítulos con tools/generar_subtitulos_whisper.py.

Este flujo es ideal para crear videos virales sin gastar créditos de IA.

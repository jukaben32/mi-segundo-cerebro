# Flujo completo con IA

1. Generar guion con Gemini (tools/generar_guion_gemini.py)
2. Generar imágenes con Gemini (tools/generar_imagenes_gemini.py)
3. Generar audio con Gemini (tools/generar_audio_gemini.py)
4. Generar subtítulos con Whisper (tools/generar_subtitulos_whisper.py)
5. Ensamblar video con FFmpeg (tools/ensamblar_video_ffmpeg.py)

Este flujo es 100% automático y solo requiere tu API Key de Gemini y los modelos instalados.

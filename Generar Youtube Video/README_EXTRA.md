# Integración con n8n

Puedes conectar estos scripts con flujos de n8n usando nodos de tipo "Ejecutar comando" o "Python". Así puedes automatizar la generación de videos desde formularios, Google Drive, o cualquier otro disparador soportado por n8n.

Ejemplo de comando en n8n:

```
python main.py "Crea un guion para un video viral de terror"
```

Puedes usar los archivos generados como outputs en otros nodos para subirlos, compartirlos o analizarlos.

# AI Marketing Strategy Suite (Claude Code)

## Overview
A comprehensive suite of 14+ skills and 5 AI agents designed to automate high-level marketing agency tasks using Claude Code. It enables user to perform deep audits, generate ad copy, build funnels, and create professional PDF reports.

## Source
- **GitHub:** [ai-marketing-claude](https://github.com/zubair-trabzada/ai-marketing-claude)
- **Transcription:** [[video-transcription-ad-strategy]]

## Key Capabilities
- **Full Marketing Audit:** Analyzes content, CRO, SEO, competition, and strategy.
- **Ad Creative Generation:** Produces copy and hooks for Meta, Google, LinkedIn, etc.
- **Funnel Optimization:** Analyzes sales funnels and proposes improvements.
- **Automated Reporting:** Generates Markdown and PDF reports.
- **Social Media Planning:** 30-day content calendars.

## Installation (Windows)
The system has been installed to `C:\Users\hp\.claude\skills` and `C:\Users\hp\.claude\agents`.

## Usage
Run the following commands in Claude Code:
- `/market audit <url>`
- `/market ads <url>`
- `/market report-pdf <url>`

## Strategy Insights
- Audiences are segmented into personas (e.g., "Professional Groomer").
- Budget is allocated based on funnel stages (Awareness, Consideration, Conversion, Retargeting).
- KPIs are tracked per platform (ROAS, CPA, CTR).

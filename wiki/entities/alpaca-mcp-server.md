## Qué es

Servidor MCP (Model Context Protocol) oficial de Alpaca. Permite interactuar con la API de Alpaca usando LLMs (Claude, ChatGPT, etc.) con lenguaje natural desde IDEs como Claude Code, Cursor, o Windsurf.

## Para qué sirve

- Ejecutar órdenes con comandos en español/inglés
- Analizar el portfolio con IA
- Consultar datos de mercado sin escribir código
- Construir estrategias describiendo la lógica

## Ejemplo de uso

En Claude Code o Cursor con MCP configurado:
```
"Compra 5 acciones de NVDA en paper trading"
"¿Cuál es mi P&L de hoy?"
"Muéstrame los datos de AAPL de los últimos 30 días"
```

## Por qué es relevante (2025-2026)

Es el puente entre LLM agents y trading algorítmico. Permite construir **AI trading agents** que operan en Alpaca sin código explícito.

## Qué es

Editor de markdown local con graph view, backlinks automáticos, y sistema de plugins. Actúa como "IDE" para el LLM Wiki: el humano navega, el LLM escribe.

## Uso en este segundo cerebro

- **Graph view**: ver la forma del wiki, hubs vs nodos huérfanos
- **Backlinks**: navegación por relaciones entre notas
- **Dataview plugin**: queries sobre frontmatter YAML
- **Marp plugin**: generar slides desde markdown
- **Web Clipper**: extensión Chrome para ingerir artículos como raw/

## Configuración recomendada

- `Settings → Files and links → Attachment folder path` = `raw/assets/`
- `Settings → Hotkeys` → bind "Download attachments" a `Ctrl+Shift+D`
- Carpeta por defecto de Web Clipper → `raw/`

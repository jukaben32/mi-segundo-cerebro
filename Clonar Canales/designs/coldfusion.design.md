# DESIGN.md - ColdFusion Style (Professional-Utility)
Este sistema define la estética "Technical Mono" y el enfoque documental de ColdFusion.

## 1. Visual Theme & Atmosphere
- **Concept:** Professional-Utility. High clarity, information-dense, investigative documentary feel.
- **Mood:** Intelligent, deep, authentic, slightly dark but transparent.
- **Surface:** Matte surfaces, no intense gloss unless it's a specific tech focus.

## 2. Color Palette & Roles
| Role | Color (Hex) | Purpose |
|------|-------------|---------|
| Background | #050505 | Primary canvas (Deep Black) |
| Secondary BG| #0A0A1A | Dark Navy surfaces |
| Primary Accent| #00F2FF | Cyber Cyan for active nodes/links |
| Text Primary | #E0E0E0 | Off-white for high readability |
| Text Mono | #80FFD0 | Subtle green for code/terminal data |

## 3. Typography
- **Headings:** Modern Sans-Serif (Outfit / Inter). Bold, wide spacing.
- **Body:** Clean Sans-Serif.
- **Mono:** JetBrains Mono for all data, code, and investigative overlays.

## 4. Visual Principles
- **Grid:** Strict geometric alignment. 8px grid system.
- **Depth:** Low elevation. Subtle shadows. Focus on 2D planes stacked in 3D space.
- **Contrast:** High. Text must pop from background.

## 5. Agent Prompt Directives
- "Use #050505 as absolute background."
- "Display data in #80FFD0 monospaced font."
- "Highlight investigative connections with #00F2FF cyber cyan."
- "Maintain a clean, documentary-style layout with technical mono aesthetics."

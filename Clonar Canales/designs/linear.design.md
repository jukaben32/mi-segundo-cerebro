# DESIGN.md - Linear Inspired (Precise Minimalism)
Este sistema emula la estética de Linear.app: precisión quirúrgica y minimalismo elegante.

## 1. Visual Theme & Atmosphere
- **Concept:** Precision Minimalism. High-end, technical, and clean.
- **Mood:** Focused, efficient, calm, and high-performance.
- **Surface:** Glassmorphism with tight blur, 1px semi-transparent borders.

## 2. Color Palette & Roles
| Role | Color (Hex) | Purpose |
|------|-------------|---------|
| Background | #0B0E14 | Deep Navy-Black |
| Secondary BG| #151921 | Lighter UI panels |
| Primary Accent| #5E6AD2 | Indigo for primary focus |
| Text Primary | #E1E1E1 | High contrast whites |
| Text Dim | #8A8F98 | Muted secondary information |

## 3. Typography
- **Headings:** Inter. Medium to semi-bold, 0.02em letter spacing.
- **Body:** Inter. Clean and readable.
- **Micro-detail:** Monospace numbers for time and technical data.

## 4. Visual Principles
- **Borders:** 1px width, #FFFFFF10 color.
- **Corners:** 6px to 8px radius (gentle rounding).
- **Shadows:** Ultra-soft, 40px blur, black shadows (no harsh edges).

## 5. Agent Prompt Directives
- "Adopt a Liquid Glass UI approach with #FFFFFF10 borders."
- "Use #5E6AD2 (Indigo) as the primary spotlight color."
- "Maintain extreme 1px precision in all visual elements."
- "Focus on #0B0E14 as the core canvas for a high-end SaaS feel."
